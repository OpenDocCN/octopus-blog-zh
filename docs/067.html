Octopus Deploy集成了流行的云服务，如亚马逊网络服务(AWS) 和微软的Azure平台，可以轻松安全地部署您的应用程序。与Amazon集成就像添加你的AWS访问密钥和秘密密钥一样简单，然而，Azure需要更多的细节，并且如何设置还不是很明显。这是通过Azure Active Directory (AD)注册的应用程序(或应用程序注册)实现的，但设置起来可能很棘手，因此我们将深入探讨并完成整个过程。

Octopus中的Azure帐户需要四个值，用于对Azure进行身份验证并与之安全交互。

  • Azure订阅ID
  • Azure AD租户ID
  • Azure AD注册的应用程序ID
  • Azure AD注册的应用程序密码/密钥

前三个值是GUIDs，最后一个是密码。这些值可以在Azure门户或通过Powershell找到。就我个人而言，我从来不记得所有的值是在哪里找到的，所以我们录制了一个视频来帮助大家！
